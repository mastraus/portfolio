The power of React in an easy-to-use web app.

To create this pomodoro timer, I was specifically responsible for all functionality. The majoritity of the design code was prepared by the Thinkful staff, but my group and I were tasked with making a working prototype. In particular, I helped support my group with state changes upon event listeners, creating components for clean code, and properly importing timer controls for the play/pause/stop functionality.
